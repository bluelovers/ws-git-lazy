# README.md



## install

```bash
yarn add @git-lazy/clone-subdir
yarn-tool add @git-lazy/clone-subdir
yt add @git-lazy/clone-subdir
```

```ts
gitCloneSubDir(`https://github.com/conventional-changelog/conventional-changelog.git`, {
	cwd: `T:\\download`,
	targetDir: `T:\\download\\test4`,
	subDir: `packages/conventional-changelog-angular`
})
```
